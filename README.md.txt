# MPDIGITAL Website

## Project Overview
This repository contains the source code and assets for the MPDIGITAL website. The site includes various sections such as a blog, portfolio, services information, and contact details.

## Directory Structure

### Assets
- **images/**
  - 5.png
  - MyLogo.svg
  - Profile.png
  - ROI.png
- **js/**
  - analytics.js - Tracks user interactions and page performance
  - calculators.js - Handles various calculation functionalities
  - script.js - Main JavaScript functionality for the site

### Pages
- Home.html - Main landing page
- blog.html - Blog articles and news
- certificates.html - Professional certifications and qualifications
- contact.html - Contact information and form
- portfolio.html - Project showcase and work examples
- services.html - Details of services offered

## Getting Started
To run this project locally:
1. Clone the repository
2. Open Home.html in your web browser

## Development
This project uses:
- HTML5 for structure
- CSS for styling (likely included within HTML files or as separate files not shown in the directory)
- JavaScript for interactive functionality
